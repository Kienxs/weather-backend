🚀 Key Features

Real-time Weather Engine: Fetches precise meteorological data including temperature, humidity, and wind speed.

Interactive Mapping: Integrated Leaflet.js to provide real-time location tracking and visual mapping.

Advanced Data Visualization: Utilizes Chart.js to render a 5-day temperature forecast with smooth, responsive line charts.

Intelligent Search: Implemented Debounce Logic on the frontend to minimize redundant API calls and optimize server resources.

Parallel Execution: Leverages Promise.all for concurrent API requests, significantly reducing initial load time.

🛠️ Technical Stack

Backend (The Engine)

Language: Java 17

Framework: Spring Boot 3.x

Communication: RESTful API Design

Build Tool: Maven

Deployment: Render (Dockerized environment)

Frontend (The Experience)

Library: React.js (Functional Components & Hooks)

Bundler: Vite (for ultra-fast development) 

Data Fetching: Axios

Geospatial: Leaflet & React-Leaflet

Charts: React-Chartjs-2
